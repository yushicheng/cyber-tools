<h2 align="center">Usage</h2>

start命令 启动mock服务:
会加载当前文件夹下所有的.mock.js文件提供mock服务
```bash
mock-server start
```

init命令 :
在当前文件下创建mock-server.config.js
```bash
mock-server init
```
